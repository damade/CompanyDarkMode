Registrar Connect

This is a web portal designed specifically for Registrar Houses; serving as repository for companies on their shareholdings and acting as a gateway for stockbrokers, shareholders and company secretaries.

The solution brings together the Registrar Houses and their clients i.e. the client companies, stockbrokers and the company shareholders under a single platform to allow for seamless information sharing, research, lodgment and tracking online.

Mindworks Payman

Mindworks Payman is mid-sized payroll application for small and medium-scale organizations

It allows for multi-company setup among other features. The system is highly parameterized to allow for flexibility. It also has the capabilities of generating bank schedules to allow for e-payment.

OMR Interface

OMR Interface is our software solution built specifically for OMR Scanners. Optical Mark Reader(OMR) is a High-Tech data input device which integrates optical and electrical technique for data capture. OMR eliminates the entire bottleneck of data input. It is a powerful tool for speedy automatic data capture. The speed is several hundreds of times faster than that of manpower; with the machine having the ability to work for a long period without break.
